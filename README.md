# Sales Performance Dashboard and Profitability Analysis

## Project Overview
This project analyzes sales data to evaluate business performance including revenue trends, profit margins, and regional sales distribution.

## Tools Used
- MySQL (Data Querying)
- Excel (Data Cleaning)
- Power BI (Dashboard Visualization)

## Key Analysis
- Revenue and profit KPI analysis
- Regional sales performance
- Category-wise revenue contribution
- Monthly sales trend analysis

## Dashboard Features
- Total Revenue KPI
- Profit Margin KPI
- Regional Sales Analysis
- Category Performance Visualization
- Monthly Sales Trend

## Dataset
Sales dataset containing transaction-level business data used for analysis.

![Dashboard](dashboard.png)

## Project Workflow

1. Data collection from sales dataset
2. Data cleaning and preprocessing in Excel
3. SQL analysis for KPIs and revenue insights
4. Power BI dashboard development
5. Visualization of revenue trends and regional performance
